![Monokaifu Theme](https://github.com/powerkaifu/monokaifu_theme/raw/master/theme_screenshot.jpg)

## 註解顏色

若想改變註解顏色，可以在 `settings.json` 中加入以下設定：

```json
	"editor.tokenColorCustomizations": {
		"[Monokaifu Theme]": {
			"comments": {
				"fontStyle": "italic",
				"foreground": "#6a9955"
			}
		}
	},
```
